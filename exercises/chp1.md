# chp1 연습문제

1.Q1
  - (a) 디바이스의 센서와 액추에이터의 역할을 비교하여 설명하세요.
  - 센서는 외부의 물리적 상태를 감지하여 전기적 신호로 변환하는 입력 역할을 하며, 액추에이터는 전기적 신호를 물리적 동작으로 바꾸어 장치를 구동하는 출력 역할을 합니다.
  - (b) 디바이스와 대시보드에서 동일한 프로그래밍 언어를 사용하지 않는 이유를 설명하세요.
  - 하드웨어 최적화가 필요한 디바이스는 저수준 언어(C/C++)를, 사용자 인터페이스와 데이터 처리가 중요한 대시보드는 고수준 언어(JS/Python)를 사용하여 각 환경의 효율을 극대화하기 위해서입니다.
  - (c) 브로커와 같은 정보 전달 전담 서버의 이점을 설명하세요.
  - 브로커(Broker)와 같은 정보 전달 전담 서버를 사용하면 디바이스와 대시보드 간의 직접적인 연결 의존성을 제거하여 시스템의 확장성과 안정성을 높일 수 있습니다.

2. Q2
  - (a) Mosquitto와 Node-RED를 별도 서버에 설치할 경우의 이점을 설명하세요.
  - 서로 다른 서버에 설치하면 시스템 부하를 분산하고, 한쪽의 장애가 다른 서비스로 전이되는 것을 막아 안정성과 확장성을 높일 수 있기 때문입니다.
  - (b) Mosquitto와 Node-RED를 운영하기 위한 적절한 컴퓨팅 환경을 기술하세요.
  - 각 프로그램의 특성에 맞춰 Mosquitto는 네트워크 입출력에 최적화된 저사양 환경이, Node-RED는 데이터 처리와 UI 실행을 위한 충분한 메모리(RAM) 환경이 적절합니다.
